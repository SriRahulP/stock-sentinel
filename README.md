# Stock Sentinel 

This project analyzes stock market data using Machine Learning and technical indicators.

## Features
- Moving Averages (7-day, 21-day)
- RSI Indicator
- Bollinger Bands
- Volatility Analysis
- Random Forest Prediction Model

## Dashboard
An interactive Excel dashboard showing:
- Price trends
- RSI signals
- Bollinger Bands
- Stock comparison table

## Model Performance
Accuracy: ~53%

## Tools Used
- Python (Pandas, Scikit-learn)
- SQLite
- Excel Dashboard

## Conclusion
This project demonstrates how ML and data analytics can assist in stock market decision-making.
